# 👔 Manual Administrativo - MIDAS Intranet

Guía para Gerentes, Directores y Administradores del Sistema.

## 1. Planificación Presupuestaria (Matriz)
Como gerente, eres responsable de crear el presupuesto de tu área.
1.  Ve a **"Planificación de Proyectos"**.
2.  Haz clic en **"Nuevo Proyecto"**.
3.  **Llenado:**
    *   Define si es OPEX (Operativo) o CAPEX (Inversión).
    *   Asigna el Trimestre (Q1, Q2, etc.).
    *   **Partidas:** Detalla cada gasto (Laptops, Licencias, Muebles). El sistema calculará el total.
4.  **Enviar:** Al finalizar, pulsa **"Enviar a Aprobación"**. El estado cambiará a `PENDIENTE_DIRECTOR`.

## 2. Flujo de Aprobación
Como Director, debes revisar los proyectos de tus gerentes.
1.  Ve a la pestaña **"Aprobaciones"** en el módulo de Planificación.
2.  Revisa el detalle de cada proyecto y sus partidas.
3.  **Acciones:**
    *   ✅ **Aprobar:** El presupuesto se activa y queda disponible para compras.
    *   ❌ **Rechazar:** Debes escribir una razón. El proyecto vuelve a estado `RECHAZADO` para corrección.

## 3. Aprobación de Compras
Cuando un empleado crea una solicitud:
1.  Recibirás una notificación en tu panel.
2.  Verifica:
    *   ¿Está ligado a un Proyecto Aprobado? (Verás el código `P-202X-X`).
    *   ¿Hay presupuesto disponible? (El sistema te avisará si `Fondos Insuficientes`).
3.  Aprueba o Rechaza la compra.

## 4. Gestión de Usuarios (Solo Admin)
Para agregar nuevos empleados:
1.  Ve a **Configuración > Usuarios**.
2.  Crea el usuario y asígnale:
    *   **Rol:** (Empleado, Gerente, Admin).
    *   **Departamento:** Crucial para el flujo de aprobación.
    *   **Supervisor:** Quien aprobará sus vacaciones y compras.

## ⚠️ Reglas de Negocio Importantes
*   **Modificación de Presupuestos:** Una vez aprobado, un presupuesto NO puede ser modificado. Debe crearse una solicitud de "Aumento de Presupuesto" (proceso externo).
*   **Gastos No Planificados:** Si apruebas una compra "Urgente" sin proyecto, afectará las métricas de desviación presupuestaria de tu departamento.
